Class Project: Wood Safe

Skills: [SolidWorks, Mechanical Design, Reverse Engineering]

This was my final project for my SolidWorks design course at WSU, ME 116. The model is reverse engineered from a one-of-a-kind handmade wood safe. Choosing this object to reverse engineer was difficult because it was not originally designed to be manufactured. None of the parts were standard sizes and I had to put a lot of effort into tolerancing to make sure there was no interference. I made all the sheet metal components using SolidWorks sheet metal tools and used toolbox components for the bolts. The mechanical components of the combination lock all interact realistically without interference, and all materials are accurate.
